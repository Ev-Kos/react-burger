# Stellar Burgers
------
## Описание проекта

Приложение космической бургерной разработаное на React.
С макетом проекта можно ознакомиться по [ссылке](https://www.figma.com/file/zFGN2O5xktHl9VmoOieq5E/React-_-%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82%D0%BD%D1%8B%D0%B5-%D0%B7%D0%B0%D0%B4%D0%B0%D1%87%D0%B8_external_link?node-id=0%3A1).

[Проект](https://Ev-Kos.github.io/react-burger/) на gh-page.

**Функционал**
- Pегистрация и авторизация пользователя
- Некоторые страницы защищены авторизацией
- Создание онлайн-заказа
- Просмотр общей и личной истории заказов
- Просмотр состава ингредиентов 
- Передача данных с API проходит через защищенное соединение с валидацией запросов


**Используемые технологии**

- React
- библиотека UI-компонентов [Яндекс.Практикум](https://yandex-practicum.github.io/react-developer-burger-ui-components/docs/readme)
- Redux
- Web Sockets
- TypeScript
